# bot-wolfram

Ask Wolfram Alpha

## Usage

```
Usage: wolfram [options] <input...>

  Alias: wf

  Ask Wolfram Alpha

  Options:

    --help  output usage information
```

## Config (config.yaml)

```yaml
wolfram:
  token: your_api_token
```

## How do I get my api token?

Go to http://products.wolframalpha.com/api/
